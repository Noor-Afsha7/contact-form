# Contact Form

A simple and responsive contact form built using HTML and CSS.  
This project is created to practice form structure, input validation, and modern UI styling.

## 🚀 Live Demo

👉 https://noor-afsha7.github.io/contact-form/

## ✨ Features

- Responsive centered form layout
- Input fields for name, email, and message
- Required field validation using HTML5
- Rounded input fields for modern UI
- Hover effect on submit button
- Clean and minimal design

## 🛠️ Tech Stack

- HTML5
- CSS3

## 📚 What I Learned

- Structuring forms using semantic HTML
- Using `<label>`, `<input>`, and `<textarea>` correctly
- Applying form validation with `required`
- Styling forms using CSS (padding, margin, border-radius)
- Adding hover effects using `:hover`
- Improving UI/UX with simple design principles
